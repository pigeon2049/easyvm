# VM Launcher (虚拟机运行器)

基于 PyQt5 与 QEMU 构建的**全绿化免安装**专属轻量级虚拟机管理器。专为 Windows 环境打造，无需复杂的环境配置即可一键下载镜像、创建多个彼此隔离的本地测试系统。

## 特色功能
- **真正的绿色便携设计**：所有操作基于相对路径映射展开，环境目录支持直接全盘拖拽移动，U 盘直插即用。
- **环境智适应机制 (Fallback)**：
  - **组件自动化部署**：应用如果检测不到本地虚拟机引擎，会在后台全自动静默从远端拉群最新版 QEMU 并快速解压挂载。
  - **无底座虚拟化突破 (TCG)**：如果你的设备 CPU 老旧或未开启 Windows Hypervisor Platform (WHPX) 硬件加速，应用将会通过纯软件架构强行拉起完整的图形指令虚拟机引擎，让你做到在任何“网吧机”级环境下都能点开即玩。
- **自带软件分发中心**：直连 TUNA（清华大学镜像源），提供从极简型 (Alpine) 到完备型 (Ubuntu / Debian) 的系统镜像选择，而且整合了**本地防多次请求缓存池**，只要下载过一次便可达到下一次拉起“秒级部署”体验。
- **进程异步隔离池**：每个运行状态都独立附着，一键并行启动或关闭多个虚拟机操作，控制台 UI 高清渲染。

## 目录结构
```text
vmapp/
├── core/                  # 核心计算逻辑：QEMU 子进程管理器、并发网络下载器、数据配置中心
├── ui/                    # 采用 PyQt5 渲染的用户控制前端交互界面
├── data/                  # （运行时自动生成）所有新建的虚拟机持久磁盘、配置以及 ISO 原片库
├── qemu/                  # （运行时自动生成）免安装 QEMU 绿化版核心依赖组件
├── launcher.py            # 主程序应用进入点
└── requirements.txt       # Python 包依赖一览列表
```

## 简单两步，开箱即用

**前置需求：**
确保你的系统安装了 Python 3，且位于 Windows 平台。

**开始使用：**
1. **克隆项目到本地并进入目录**:
```bash
git clone https://github.com/pigeon2049/easyvm.git
cd easyvm
```

2. **安装 UI 界面开发依赖引擎**:
```bash
pip install -r requirements.txt
```

3. **启航应用**:
```bash
python launcher.py
```
